SCA-IA
===
SCA-IA point cloud registration and ICP to solve RT matrix
Developed By VS2013update5 and PCL(Point Cloud Library)

Command Line
---

use program in cmd:

```
    >> PointCloudRegistration.exe source.pcd target.pcd
```
